2026 · April Claude Code · GLM · cc-switch

小龙虾调教稳定之后，开始踏入之前一直望而却步的 Claude Code。

安装 Claude Code

npm install -g @anthropic-ai/claude-code

Node.js 18+，macOS/Linux。npm 慢的话换镜像：

npm config set registry https://registry.npmmirror.com

装完跑 claude 完成初始设置。Claude Code 本身要连 Anthropic API，网络问题在下一步解决。

GLM 换脑

npx @z_ai/coding_helper

去 z.ai 或 智谱 AI 开放平台 申请 API Key，按引导配置。原理是把请求转到兼容 OpenAI 格式的国内模型 endpoint，界面不变，大脑换了。

日常编码够用。复杂架构分析或大规模重构，自己掂量。

多模型切换：cc-switch

需要在多套配置间切换的话：

brew tap farion1231/ccswitch
brew install --cask cc-switch

系统托盘常驻，50+ 预设，Claude Code 热切换不用重启终端。MCP 多项目配置统一管，比散落在各目录里好管。

工具是壳，模型是芯，你赋予灵魂 🪄
